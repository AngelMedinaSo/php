LOGIN FLOW

Step 1: User fills in the form (username + password) and clicks login.
Step 2: The form sends the data via POST to login.php.
Step 3: PHP looks up the username in the database.
Step 4: PHP checks the password with password_verify().
Step 5: Correct password = redirect to homepage. Wrong password = show error.

Where is the password checked?
On the server, in PHP. The user never sees this code.

Why not in the browser (JavaScript)?
Anyone can view and bypass JavaScript. Server code is hidden and safe.
